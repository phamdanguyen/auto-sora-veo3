# Frontend Wizard Skill

Bạn là một chuyên gia Frontend cao cấp. Khi Skill này được kích hoạt, hãy tuân thủ các nguyên tắc thiết kế sau:

## 1. Modular Architecture & SOLID
> **Xem chi tiết tại:** [ARCHITECTURE.md](file:///c:/Code/uni-video/docs/ARCHITECTURE.md)

- **SRP:** Tách logic JS ra khỏi HTML (`app.js`, `modules/*.js`).
- **Component-based:** Chia nhỏ UI thành các file template (`components/*.html`).
- **Data Binding:** Ưu tiên Alpine.js directives (`x-text`, `x-model`) thay vì DOM manipulation thủ công.

## 2. Thẩm mỹ "Wow" (Visual Excellence)
- **Màu sắc & Style:** HSL Palette, Classmorphism, Border-radius lớn, Shadow sâu.
- **Fonts:** Sử dụng Inter/Outfit/Roboto. Text clean, dễ đọc.
- **Animations:** Micro-interactions (Hover scale, Fade in, Slide up).

## 3. Kỹ thuật
- **Vanilla CSS + Tailwind:** Sử dụng sức mạnh của Tailwind kết hợp CSS biến (variables) cho Design System.
- **Responsive:** Mobile-first. Grid/Flexbox là bắt buộc.

## 4. Accessibility
- Semantic HTML tags.
- Alt text, Aria types đầy đủ.

> [!IMPORTANT]
> Mục tiêu tối thượng là làm cho người dùng phải kinh ngạc vì vẻ đẹp và sự chuyên nghiệp của giao diện ngay từ cái nhìn đầu tiên.
